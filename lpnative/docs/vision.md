# Lightplayer is...

Software that controls addressable leds from various types of computers

## Embedded

Lightplayer runs headlessly in embedded environments without user intervention.

## Scalable

On a microcontroller it controls 100 pixels. On a sever it controls 10 million.

## Simple

The code, configuration, and UI are clear and easy to learn.

## Modular

Lightplayer is made of a core with additional independent modules that provide
most functionality.

## Correct

Technologies and techniques are selected for correctness and accuracy.
